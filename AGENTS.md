# Course Builder Harness

이 프로젝트는 Codex가 온라인 강의용 커리큘럼, 교안, 퀴즈, 실습과제를 일관된 방식으로 설계하도록 돕는 작업 지침 모음이다.

## 목적

- 사용자 요청을 교육과정 설계 작업으로 해석한다.
- 필요한 산출물을 `_workspace/` 아래 표준 파일명으로 정리한다.
- 학습목표, 교안, 평가, 실습이 서로 정렬되도록 유지한다.
- 기존 산출물이 주어지면 재사용하고, 부족한 단계만 새로 작성한다.

## 표준 구조

```text
AGENTS.md
.codex/
  agents/
    curriculum-designer.toml
    content-writer.toml
    quiz-maker.toml
    lab-designer.toml
    course-reviewer.toml
.agents/
  skills/
    course-builder/SKILL.md
    learning-design/SKILL.md
    assessment-engineering/SKILL.md
    lab-scaffolding/SKILL.md
_workspace/
  00_input.md
  01_curriculum.md
  02_lesson_plans.md
  03_quizzes.md
  04_labs.md
  05_review_report.md
```

## 작업 모드

요청 범위에 따라 아래 모드 중 하나를 고른다.

1. 풀 파이프라인
   - 커리큘럼, 교안, 퀴즈, 실습, 리뷰까지 모두 생성한다.
2. 커리큘럼 모드
   - `01_curriculum.md`와 필요 시 `05_review_report.md`만 생성한다.
3. 교안 모드
   - 기존 커리큘럼을 기반으로 `02_lesson_plans.md`를 작성하고 검토한다.
4. 퀴즈 모드
   - 기존 커리큘럼과 교안을 기반으로 `03_quizzes.md`를 작성하고 검토한다.
5. 실습 모드
   - 기존 커리큘럼과 교안을 기반으로 `04_labs.md`를 작성하고 검토한다.

## 공통 워크플로우

1. 요청을 구조화한다.
   - 주제
   - 대상 학습자 수준
   - 총 학습 시간 또는 과정 규모
   - 실습 환경, 언어, 도구
   - 기존 산출물 유무
2. 입력을 `_workspace/00_input.md`에 정리한다.
3. 기존 파일이 있으면 덮어쓰지 말고 기준 문서로 삼는다.
4. 커리큘럼이 없으면 먼저 `01_curriculum.md`를 만든다.
5. 교안, 퀴즈, 실습은 커리큘럼 이후에 진행한다. 가능하면 병렬로 처리한다.
6. 마지막에 리뷰 보고서로 커버리지와 누락을 검증한다.
7. 최종 응답에서는 생성한 파일, 주요 가정, 남은 리스크를 짧게 요약한다.

## 산출 기준

- 학습목표는 측정 가능한 행동 동사로 작성한다.
- 학습목표에는 추적 가능한 ID를 붙인다.
- 기본적으로 한 레슨은 15~30분, 한 모듈은 3~7개 레슨으로 설계한다.
- 이론:실습 기본 비율은 6:4로 두되, 조정 시 이유를 적는다.
- 전문 용어는 첫 등장 시 정의하고 용어집에 남긴다.
- 모든 퀴즈 문항은 특정 학습목표와 블룸 수준에 매핑한다.
- 실습은 따라하기에서 자율 프로젝트로 점진적으로 어려워져야 한다.
- 리뷰는 반드시 교안, 퀴즈, 실습의 학습목표 커버리지를 교차 검증한다.

## 기본 가정 규칙

- 사용자가 수준을 명시하지 않으면 초급 학습자를 기본값으로 둔다.
- 언어나 환경이 없으면 일반적으로 접근성이 높은 도구를 우선 제안한다.
- 최신 사실이나 도메인 정보가 필요하면 검증 후 사용하고, 확인 기준 시점을 결과에 적는다.
- 불명확한 정보는 임의로 숨기지 말고 "가정" 섹션에 기록한다.

## 출력 규칙

- 사용자의 언어를 우선 따른다. 별도 지시가 없으면 한국어로 작성한다.
- Markdown을 사용하고 바로 저장 가능한 완성형 문서로 작성한다.
- 불완전한 단계가 있으면 완료로 표시하지 않는다.
- 리뷰에서 치명적 누락이 남아 있으면 수정 필요 상태를 명시한다.

## 범위 제외

아래 작업은 이 하네스의 기본 범위가 아니다.

- LMS 구축 또는 운영 자동화
- 동영상 촬영, 편집, 자막 제작
- 수강생 운영, 결제, 수료증 발급
- 실제 강의 플랫폼 배포
